\cvsection{Beyond Work}

I've been programming since I was 13, starting with terrible RPGMaker Ruby scripts before deciding C++ would be an excellent second language. I never quite managed to kick the habit. I still tinker and learn in my own time: hobbyist game development with C\#, Godot, and Unity, and a webapp for the card game Riftbound built as a monorepo with extensive integration and contract testing. I also write fiction and have run a weekly tabletop group since COVID, so storytelling tends to find its way into most of what I do. You'll usually find me nestled reading or working at my local caf\'e, or running sessions and playing card games at hobby shops.
