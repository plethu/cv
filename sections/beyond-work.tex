\cvsubsection{Current Projects}

\textbf{Recite} is my current open-source focus: a Rust dialogue compiler, runtime, and toolchain for narrative-heavy games. The interesting work is keeping authored prose pleasant while enforcing deterministic traversal, stable localisation IDs, schema-checked effect requests, and validation boundaries before runtime.

\textbf{Candlelit Sanctum} is a Riftbound companion app across web, mobile, API, and shared packages. The interesting work is turning a fast-moving card game into reliable software: catalogue ingestion, searchable card modelling, deck validation, typed client/API contracts, background sync, media handling, and shared tests to keep game-rule logic honest.

I've been programming since I was 13 and never quite stopped tinkering. I also write fiction and have run a weekly tabletop group since COVID, so storytelling tends to find its way into most of what I do. You'll usually find me reading or working at my local caf\'e, or running sessions and playing card games at hobby shops.
